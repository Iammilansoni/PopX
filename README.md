# Educase India - UI Assignment

This is the **UI Assignment** for **Educase India**, built using **Next.js** and **TailwindCSS**. The project is designed to be responsive, user-friendly, and visually appealing.

## 📌 Tech Stack

- **Next.js** (15.1.0)
- **React** (19)
- **TailwindCSS** (3.4.17)
- **TypeScript** (5)
- **React Hook Form**
- **Radix UI Components**
- **Recharts**
- **Embla Carousel**
- **Zod Validation**
- **Lucide Icons**
- **PostCSS**
- **Sonner for Toast Notifications**

---

## 🚀 Getting Started

### 1️⃣ Prerequisites

Ensure you have the following installed:

- **Node.js** (Latest LTS Recommended)
- **npm** or **yarn**

### 2️⃣ Installation

Clone the repository and install dependencies:

```sh
git clone https://github.com/Iammilansoni/PopX.git
cd PopX
npm install

3️⃣ Running the Project

To start the development server, run:

npm run dev

Open http://localhost:3000 in your browser.

4️⃣ Build for Production

npm run build
npm run start
